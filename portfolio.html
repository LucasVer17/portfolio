Geolocator

Este projeto exibe a localização do usuário em um mapa interativo usando Leaflet e a API do OpenWeather. O usuário pode visualizar sua localização atual ou pesquisar uma cidade, que será marcada no mapa.

Bookmark website

Este projeto é um Bookmark Website simples, onde o usuário pode adicionar e visualizar links com seus respectivos nomes. O layout foi feito com Tailwind CSS.
